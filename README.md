# Location-Based Messaging App

A hybrid Android application with **Spring Boot** backend that enables users to post and discover location-specific messages. Think of it as a digital "post-it wall" where anyone passing through a location can read what others have shared.

## Overview

This app allows users to:
 - Leave messages tied to specific geographic locations.
 - Discover nearby posts left by others.
 - Communicate even without internet using Wi-Fi Direct peer-to-peer sharing.
